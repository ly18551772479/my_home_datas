# 方法

## 1. 分类
而在Python中，方法分为三类:
* 实例方法
* 类方法
* 静态方法

代码如下：

```python

class Test(object):

	#实例方法
	def instancefun(self):
		print("InstanceFun")
		print(self)

	#类方法
	@classmethod
	def classfun(cls):
		print("ClassFun")
		print(cls)

	#静态方法
	@staticmethod
	def staticfun():
		print("StaticFun")
```

+ 实例方法 - 隐含的参数为类实例self
+ 类方法 - 隐含的参数为类本身cls
+ 静态方法 - 无隐含参数，主要为了类实例也可以直接调用静态方法。
+ 类名可以调用类方法和静态方法，但不可以调用实例方法
