uWSGI操作

安装uWSGI

pip install uwsgi

配置uWSGI

  • 在项目目录下创建uwsgi.ini文件

  • uwsgi.ini文件内容

    [uwsgi]
#使用nginx连接时使用，Django程序所在服务器地址
#socket=172.20.10.3:8080
#直接做web服务器使用，Django程序所在服务器地址
http=172.20.10.3:8080
#项目目录
chdir=/Users/zhangjie/Desktop/DF/dailyfresh
#项目中wsgi.py文件的目录，相对于项目目录
wsgi-file=dailyfresh/wsgi.py
# 进程数
processes=4
# 线程数
threads=2
# uwsgi服务器的角色
master=True
# 存放进程编号的文件
pidfile=uwsgi.pid
# 日志文件，因为uwsgi可以脱离终端在后台运行，日志看不见。我们以前的runserver是依赖终端的
daemonize=uwsgi.log
# 指定依赖的虚拟环境
virtualenv=/Users/zhangjie/.virtualenvs/py3_django

启动

uwsgi --ini uwsgi.ini

查看

ps ajx|grep uwsgi

测试

  • 在浏览器中输入uwsgi.ini文件中配置的Django程序所在地址
  • 比如：访问天天生鲜的动态页面
    • 172.16.1.66:8080/index

停止

uwsgi --stop uwsgi.pid
